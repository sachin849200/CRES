# Class Representative Election System (CRES)

A complete secure online voting system built with PHP & MySQL

## Features
- Student & Admin login
- Create multiple elections
- Add candidates with photos
- One student = One vote (duplicate blocked)
- Live results with winner highlight
- Download official PDF result
- Nuclear Reset option

## How to Run
1. Install XAMPP
2. Put all files in `htdocs/cres`
3. Import `cres_db.sql` in phpMyAdmin
4. Run: `composer install` in terminal (inside project folder)
5. Open: http://localhost/cres
